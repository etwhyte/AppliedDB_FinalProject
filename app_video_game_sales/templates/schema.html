Database Schema

This project uses a normalized relational database created from the raw my_vgsales.csv file.

  • games: title, release year, regional sales, global sales, and foreign keys
  • platforms: platform lookup table
  • genres: genre lookup table
  • publishers: publisher lookup table
  • vg_sales_raw: staging table that keeps the original flat CSV structure
